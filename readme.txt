== Description ==
An awesome plugin that help WordPress developers to develop their themes faster than ever.

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "WP Dev Helper"
3. **Activate** WP Dev Helper from your Plugins page
4. **Click** on the new menu item "Developers" and start using!
